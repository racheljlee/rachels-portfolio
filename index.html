Weather Wander

Traveling to a Desired Temperature

Weather Wander is a web app that allows users to find a destination anywhere in the US that matches a specific temperature. After choosing a city, the web app also helps you with providing the next 5-day weather forecast and bookable local hotels nearby!

The website uses the FourSquare API, which allowed us to create a four-point polygon of coordinates to encompass all of the US' cities. It also utilizes the OpenWeather API to pull a 5-day weather forecast of the chosen city. The FourSquare API also allowed us to pull neary hotels in the city for the user to choose. We utilized Firebase to create a temporary "database" to store the user's history of cities.

I contributed greatly to the front-end/UI design/development and jQuery interactions. I also assisted with retrieving data from the APIs and displaying them visibly.

I also took on a leadership role for this project, by guiding the team through the process of brainstorming and ultimately choosing a project idea to see through to the end.

I did the logo design for the project as well.

Technologies: HTML, CSS, Bootstrap, Javascript, jQuery, Firebase, APIs
